Store Application 🛒
A simple and user-friendly store application built with Java (OOP) and HTML/CSS/JS.
📋 Project Overview
This project is a store application that allows users to browse products by category, select quantities, and automatically calculate the total price before placing an order.
✨ Features

Browse products by category (Electronics, Clothing, Food)
Select product quantity
Automatic total price calculation
Shopping cart management
Order confirmation system

🗂️ Project Structure

Product.java — Stores product name, price, category, and quantity
Category.java — Groups products into categories
Cart.java — Manages selected items and calculates total
Order.java — Confirms and displays the final order
Main.java — Main entry point of the application
store_app.html — Frontend UI built with HTML, CSS, and JavaScript

🛠️ Technologies Used

Java (Object-Oriented Programming)
HTML5
CSS3
JavaScript

🚀 How to Run
Java:

Open the project in VS Code
Open Main.java
Click the Run button

HTML:

Open store_app.html in VS Code
Right-click → Open with Live Server
